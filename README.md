## ðŸ“‹ Deskripsi Project

Music Explorer adalah aplikasi React untuk mencari dan menjelajahi musik, album, dan artis menggunakan iTunes Search API. Aplikasi ini memungkinkan pengguna untuk:
- Mencari musik, album, dan konten media lainnya
- Mendengarkan preview audio dari track
- Membuat dan mengelola playlist pribadi
- Menyimpan playlist secara lokal (localStorage)

## âœ¨ Fitur Utama

1. **Form Pencarian** - 5 input dengan HTML5 validation (keyword, media type, country, limit, explicit filter)
2. **Tabel Hasil** - Menampilkan artwork, track name, artist, album, price, release date
3. **Audio Preview Player** - Play/pause preview musik langsung di tabel
4. **Playlist Builder** - Tambah/hapus track, simpan di localStorage
5. **Sorting** - Urutkan berdasarkan release date atau price
